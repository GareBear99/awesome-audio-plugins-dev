# Image Sources for Link Page Screenshots

Every PNG in `LINK_PAGES/assets/` is currently a placeholder card. This document maps each plugin to the official source where a real screenshot should come from.

**Automated:** Run `bash scripts/download_images.sh` from the repo root to download images that have known direct URLs.

**Manual:** For plugins marked "Manual" below, visit the official page in a browser, save or screenshot the product image, and save as `LINK_PAGES/assets/{name}.png`.
