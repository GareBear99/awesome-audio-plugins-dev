# Banned

This page tracks entries that were **rejected and disqualified** from this awesome list.

Unlike [DECLINED.md](DECLINED.md), this page is for entries that should **not** be reconsidered unless there is an exceptional reason. This list intentionally does **not** include outbound links.

Related pages:
- [DECLINED.md](DECLINED.md)
- [REEVALUATION.md](REEVALUATION.md)
- [ISSUES_FOR_REEVALUATION.md](ISSUES_FOR_REEVALUATION.md)

---

## Why Something Gets Banned

Entries may be banned for reasons such as:

- malicious downloads or suspicious behavior
- fake, deceptive, or misleading claims
- spam submissions or abusive self-promotion
- repeated dishonesty during review
- impersonation or stolen project identity
- hidden paywalls presented as free
- persistent broken or bait-and-switch links
- policy-breaking or clearly harmful content
- repeated low-quality resubmission after clear rejection reasons

## Banned Entries

> This section should list names only, with reasons, and no outbound links.

### Template

#### Entry Name
- **Type:** Plugin / Bundle / Framework / Resource / Ecosystem Project
- **Status:** Banned
- **Reason(s):** Malicious / Deceptive / Spam / Impersonation / Misleading free claim / Broken bait-and-switch / Harmful content / Repeated abusive resubmission
- **Summary:** Short explanation of why it was banned.
- **Date Banned:** YYYY-MM-DD
- **Reconsideration:** No / Exceptional cases only

---

## Moderation Notes

- Use bans carefully and document the reason clearly.
- Bans protect the trust and usefulness of the list.
- If an entry was merely incomplete or weak, it belongs in **DECLINED.md**, not here.
