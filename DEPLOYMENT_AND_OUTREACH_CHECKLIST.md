# Repository Launch & Outreach Checklist

This checklist is intended for maintainers preparing the repository for publication, review, and external linking.

## Repository validation

- [ ] README renders cleanly
- [ ] tables render correctly
- [ ] no broken markdown links
- [ ] all category pages open properly
- [ ] badges render correctly, if used
- [ ] no duplicate section headings remain
- [ ] repository description and topics are configured on GitHub
- [ ] social preview image is configured on GitHub

## Internal ecosystem linking

Add natural links to this repository from:

- [ ] FreeEQ8 README
- [ ] TizWildinEntertainmentHUB
- [ ] Portfolio repository
- [ ] related plugin repositories where the link adds context

## External discovery

- [ ] identify relevant awesome lists or repository indexes
- [ ] submit the repository where appropriate
- [ ] contribute outward to related repositories where useful
- [ ] ensure the repository has a clear public-facing description before outreach

## Ongoing maintenance

- [ ] review links periodically
- [ ] update outdated entries
- [ ] remove duplicate or weak listings
- [ ] expand category pages only when the structure remains readable
