# Contributing

Thanks for helping improve this list.

## What belongs here

Add projects that are at least one of the following:
- high quality free audio plugins
- open-source plugin or DSP projects
- serious plugin development frameworks
- learning resources that genuinely help audio developers
- communities, references, or tooling with clear value

## Basic rules

- Keep descriptions short.
- Be accurate.
- Do not use marketing fluff.
- Prefer official project pages or official repositories.
- Put things in the most specific category possible.
- Avoid duplicates.

## Entry format

```md
- [**Project Name**](https://example.com) — One-sentence description.
```

## Pull request checklist

Before opening a PR, make sure:
- the link works
- the item is in the right section
- the description is concise
- the formatting matches the rest of the file
- the item is not already listed

## Start Here ecosystem categories

This repo accepts resources that support the audio-plugin/dev lane. For broader artist-platform entries, use `awesome-music-platforms`; for Python DSP/MIR/research entries, use `awesome-python-audio-science`.

Accepted here:

- free audio plugins
- open-source DSP/plugin repositories
- JUCE / VST3 / AU / CLAP / LV2 resources
- audio DSP learning resources
- sample-pack and sound-resource links with production/developer relevance
- ecosystem router links that help users find plugins, packs, deconstructed loops, visualizers, or release surfaces

Descriptions should stay short, neutral, and useful to strangers.
