# Contributing

Thanks for helping improve this list.

## What belongs here

Add projects that are at least one of the following:
- high quality free audio plugins
- open-source plugin or DSP projects
- serious plugin development frameworks
- learning resources that genuinely help audio developers
- communities, references, or tooling with clear value

## Basic rules

- Keep descriptions short.
- Be accurate.
- Do not use marketing fluff.
- Prefer official project pages or official repositories.
- Put things in the most specific category possible.
- Avoid duplicates.

## Entry format

```md
- [**Project Name**](https://example.com) — One-sentence description.
```

## Pull request checklist

Before opening a PR, make sure:
- the link works
- the item is in the right section
- the description is concise
- the formatting matches the rest of the file
- the item is not already listed
