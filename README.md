# Awesome Audio Plugin Development

A curated list of **free audio plugins**, DSP tools, and resources for
building audio plugins (VST3, AU, CLAP, LV2).\
Focused on **plugin developers, DSP engineers, and open‑source music
technology**.

Inspired by the awesome-list movement.

------------------------------------------------------------------------
