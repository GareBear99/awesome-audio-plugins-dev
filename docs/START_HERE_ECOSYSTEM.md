# Start Here Ecosystem Alignment

This repository is the **Start Here** technical list for the GareBear99 / TizWildin audio ecosystem.

## Role

`awesome-audio-plugins-dev` is the first technical stop for visitors who want:

- free audio plugins
- open-source DSP references
- JUCE / VST3 / AU / CLAP / LV2 development resources
- plugin development frameworks
- DSP learning resources
- sample-pack and creator-resource links
- routing into the wider TizWildin public `.io` ecosystem
